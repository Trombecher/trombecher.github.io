# New Colors

The ultimate color is yet to be built and I think I am very close!

## Requirements

- Multiple Colors
- Distinct color pickers for each color
- Live preview of website with those colors + custom bindings & dark mode
- Color picker include RGB (sRGB) (0..=255, 0..=1), Display P3, HSL (sRGB), HSB (sRGB), OKLAB, CMYK (sRGB)
- Contrast checker
